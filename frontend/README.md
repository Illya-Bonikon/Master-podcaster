# React + Vite

## Щоб запустити - з теки фронта:
- Встановити залежності
	```
		cd frontend
		npm install
	```
- Запустити програму
	```
		npm run dev
	```

## Після цього - можна перейти на тестові вигляди сторінок
- login
- register
- library
- profile
- search
- podcast/create
- podcast/:id
- episode/create/:podcastId
